# Prompt 1: Hook Thread (Authority + Teaching)

**Format:** Bold claim + numbered insights  
**Goal:** Establish expertise, build followers, drive DMs  
**Length:** 10-15 tweets  

---

## The Prompt

```
You are an expert Twitter/X thread writer specializing in high-engagement authority threads.

Write a complete Twitter/X hook thread for the following:

**Topic:** [TOPIC — what is the thread about?]
**Your angle/expertise:** [ANGLE — what do you know about this that most people don't?]
**Target audience:** [AUDIENCE — who are you writing for? e.g., "SaaS founders," "IT directors," "freelance designers"]
**Goal:** [GOAL — followers, leads, product signups, brand awareness, debate?]
**Voice:** [VOICE — professional, casual, technical, conversational, or a mix]
**Optional context:** [Any personal story, specific example, or data point you want to include]

---

Write a hook thread with the following structure:

**Tweet 1 (The Hook):**
- Use the "bold claim + number" formula: "I [did X thing]. Here are [N] things I learned: 🧵"
- OR: "[Number] [things/rules/lessons] about [topic] that [most people don't know / changed how I think / I wish I knew earlier]: 🧵"
- Must be under 280 characters
- Must make the reader feel: "I NEED to read the rest of this"
- Hook score: rate this tweet 1-10 and explain why it works (or what would make it stronger)

**Tweets 2 through [N-1] (The Insights):**
- Number each insight (2/, 3/, 4/, etc.)
- One insight per tweet
- Start each with a short, punchy statement (the insight itself)
- Follow with 1-3 sentences of explanation or example
- Every tweet must be ≤280 characters
- Vary sentence length — short punches followed by longer explanation
- Use line breaks strategically (Twitter rewards scannable content)

**Tweet [N] (The Close + CTA):**
- Summarize the thread value in one sentence
- Give a clear CTA: follow, DM a specific word, reply with their situation, or link to a resource
- Do NOT say "like and retweet if this helped" — too generic
- Make the CTA feel earned (they read the whole thread, now give them the next step)

**After the thread, provide:**
- **Engagement tips:** 2-3 specific things that will boost this thread's performance (best time to post, whether to add images, whether to pin it, etc.)
- **Repurposing kit:** How to turn this thread into (1) a LinkedIn post, (2) an email newsletter section, (3) a blog post outline
- **Hashtag recommendations:** 3-5 hashtags if relevant to this topic (skip if the audience is in a niche that doesn't use hashtags)

**Format rules:**
- Each tweet on its own line, preceded by "Tweet [N]:" label
- Character count in brackets after each tweet: [247 chars]
- Flag any tweet over 280 chars with ⚠️
- Use plain text — no markdown within the tweets themselves (Twitter doesn't render it)
```

---

## When to Use This Format

- You have hard-won expertise in a topic
- You want to teach something without a product pitch
- You're building in public and want followers/credibility
- You have a framework or system to share
- You want to position yourself for a new role or market

---

## Tips for a Great Hook Thread

1. **The number matters.** "7 things" outperforms "lessons I learned" by 40%+. Use a number.
2. **Don't bury the hook.** Tweet 1 is everything. If it doesn't stop the scroll, no one reads tweet 2.
3. **Shorter tweets perform better.** Aim for 150-220 chars per tweet in the body. Tweet 1 can be longer.
4. **End with a question or action.** "What am I missing?" gets replies. Replies boost reach.
5. **Post at 7-9 AM your audience's timezone.** Threads decay fast — early engagement = algorithm boost.
