# Prompt 2: How-To Thread (Step-by-Step Education)

**Format:** Result promise → numbered steps → soft product/service CTA  
**Goal:** Educate audience, drive product signups, establish expertise  
**Length:** 8-12 tweets  

---

## The Prompt

```
You are an expert Twitter/X thread writer specializing in step-by-step instructional threads.

Write a complete how-to thread for the following:

**Topic:** [TOPIC — what process, skill, or system are you teaching?]
**The result:** [RESULT — what will the reader achieve after following these steps?]
**Target audience:** [AUDIENCE — who are you writing for? e.g., "early-stage founders," "freelancers," "marketing managers"]
**Your credibility:** [CREDIBILITY — why are you qualified to teach this? One sentence is fine.]
**Product/service to soft-pitch (optional):** [If you have a tool, course, or service that makes this easier, mention it here]
**Voice:** [VOICE — professional, casual, technical, conversational]

---

Write a how-to thread with the following structure:

**Tweet 1 (The Result Promise):**
- Lead with the result, not the process: "Here's how to [achieve RESULT] in [timeframe or steps]:"
- Include a specificity signal: time saved, money earned, outcome improved
- Must make the reader feel: "I need this result. Let me read the steps."
- Must be under 280 characters
- Hook score: rate 1-10 with brief explanation

**Tweets 2 through [N-1] (The Steps):**
- Number each step (Step 1:, Step 2:, etc. — OR use 2/, 3/, 4/ notation)
- One step per tweet
- Format: [Step name/verb] → [2-3 sentences explaining HOW to do it]
- If a step has a common mistake, include: "Most people [wrong approach]. Instead, [right approach]."
- Every tweet ≤280 characters
- No jargon that isn't defined — if you use a term, briefly define it

**Tweet [N] (The Payoff + Soft CTA):**
- Acknowledge that this is the hard way
- If you have a tool/service: "If you want the shortcut: [describe what you offer in 1 sentence]. [CTA: DM me X / link below / etc.]"
- If no product: "Save this thread. The next time you [face this situation], you'll thank yourself."
- Do NOT hard-sell — this is a value thread, not a pitch thread

**After the thread, provide:**
- **Implementation shortcut:** One sentence on the fastest way to get started with Step 1
- **Common failure mode:** The one step where most people give up or go wrong
- **Repurposing kit:** How to turn this into (1) a LinkedIn carousel post, (2) a YouTube tutorial outline, (3) an email autoresponder sequence (3 emails covering the steps)

**Format rules:**
- Each tweet on its own line with "Tweet [N]:" label
- Character count in brackets after each tweet: [247 chars]
- Flag tweets over 280 chars with ⚠️
- Plain text only within tweets
```

---

## When to Use This Format

- You have a repeatable process or system
- You want to drive signups for a tool or course
- You're launching something and want to educate before you pitch
- You want to be the go-to resource for a specific skill

---

## Tips for a Great How-To Thread

1. **Start with the result, not "I'm going to show you."** Nobody clicks for the promise of learning — they click for the result.
2. **Be specific with steps.** "Research your competitors" is not a step. "Go to G2.com, filter by your category, read the 3-star reviews" is a step.
3. **Acknowledge the hard parts.** "This step takes 2 hours the first time. After that, 20 minutes." builds credibility.
4. **The soft CTA is the high-leverage move.** Don't sell in a how-to thread — build trust, then DM follow-ups convert.
5. **Add a screenshot or diagram if possible.** Visual + text threads get 2-3x more engagement than text-only.
