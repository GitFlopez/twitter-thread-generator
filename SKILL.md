## The 4 Prompts

### Prompt 1: Hook Thread (Authority + Teaching)
Generates a 10-15 tweet thread that establishes expertise. Opening tweet uses the "bold claim + number" formula. Each tweet delivers one insight. Closes with a CTA to follow/DM/subscribe.

**Best for:** Knowledge dump, "things I learned from X" format, system reveals, frameworks

### Prompt 2: How-To Thread (Step-by-Step Education)
Generates a 8-12 tweet instructional thread. Tweet 1 = the result promise. Tweets 2-10 = the steps (one per tweet). Final tweet = "if you want the shortcut, here's what I built."

**Best for:** Tutorials, process reveals, skill teaching, product education

### Prompt 3: Contrarian Thread (Opinion + Debate)
Generates a 6-10 tweet thread built for engagement. Tweet 1 = a statement most people disagree with. Tweets 2-8 = the argument. Tweet 9 = the nuance/caveat. Tweet 10 = the ask for debate.

**Best for:** Hot takes, "hot take but...", industry myths debunked, contrarian positioning

### Prompt 4: Story Thread (Narrative + Emotion)
Generates a 8-15 tweet personal story thread. Tweet 1 = the moment/event hook. Middle = the arc (struggle, turning point, lesson). Final tweet = the takeaway + soft CTA.

**Best for:** Case studies, career lessons, wins/losses, "here's what I was wrong about"
